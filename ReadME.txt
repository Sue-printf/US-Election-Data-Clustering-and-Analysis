How to run the Jupyter Notebook

* Have the latest python version installed on your computer.

* Open your command prompt and install needed libraries using the following command:pip install pandas scikit-learn matplotlib requests beautifulsoup4

* Download and install Anaconda's latest version. You can use the following link: https://www.anaconda.com/download

* After successful installation of Anaconda, launch jupyter notebook.

* From the dashboard, navigate to the notebook file(us_presidential_election_analysis.py) and click on it  to open and run it.

* You can run each cell by navigating through them using your mouse or arrow keys. Select a cell and click the run button in the toolbar.

* Continue running the cells in the notebook sequentially until you have executed all the necessary code cells.
